# 質問フロー設計

## 目的

Concur初心者が経費タイプ名を知らなくても、行動ベースの質問に答えることで適切な経費タイプへたどり着けるようにする。

## 基本方針

- 経費タイプ名を直接聞かない
- 利用者の行動を聞く
- 質問と判定ルールを分離する
- 会社ごとに質問内容を変更できるようにする

## 初期フロー

### Q1：何をしましたか？

| 選択肢       | value    | 次の質問    |
| ------------ | -------- | ----------- |
| 移動した     | travel   | q-transport |
| 宿泊した     | hotel    | result      |
| 食事した     | meal     | later       |
| 物を買った   | purchase | later       |
| 判断できない | unknown  | later       |

### Q2：移動手段は？

| 選択肢     | value | 判定結果         |
| ---------- | ----- | ---------------- |
| 電車・バス | train | 電車・近隣交通費 |
| タクシー   | taxi  | タクシー         |

## 最初に対応する範囲

- 電車・バス
- タクシー
- ホテル

## 今後追加する範囲

- 新幹線
- 飛行機
- 会食
- 備品購入
- 判断できない場合の自由入力
